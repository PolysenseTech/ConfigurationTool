3.杀毒软件Windows Defender误报解决方法

通过官网下载配置工具ConfigurationTool时，在部分机器上会被杀毒软件Windows Defender误报病毒，对程序进行干扰以及删除，在此特别声明：配置工具仅用于对设备参数进行配置，不会对客户计算机进行任务非法破坏行为。虽然我们理解杀毒软件的行为，但是也实属无奈，我们会在杀毒软件误报后，第一时间提交申诉（国内杀毒软件一般3个工作日解除，国外杀毒软件15个工作日左右）。

  • 步骤1、在 Windows 右下角找到一个盾牌图标，然后点击。
  • 步骤2、关闭实时保护、云提供的保护后。点击“添加或删除排除项”,将桌面文件夹（或其他文件夹）设置为排除项
  • 步骤3、从官网下载配置工具，并将下载好后的配置工具复制到上一步添加排除项的文件夹中。
  • 步骤4、重新打开步骤1中的实时保护、云提供的保护。 完成

3.Solution to false alarms in antivirus software Windows Defender

When downloading the Configuration Tool from the official website, some machines may be falsely reported as viruses by the antivirus software Windows Defender, causing interference and deletion of the program. It is hereby declared that the configuration tool is only used to configure device parameters and will not engage in illegal task destruction behavior on client computers. Although we understand the behavior of antivirus software, it is also helpless. We will submit an appeal as soon as possible after the antivirus software reports a false alarm (domestic antivirus software usually takes 3 working days to resolve, while foreign antivirus software takes about 15 working days).

  • Step 1: Find a shield icon in the bottom right corner of Windows and click on it.
  • Step 2: Turn off real-time protection and cloud provided protection. Click "Add or Remove Exclusions" to set the desktop folder (or other folder) as an exclusion item
  • Step 3: Download the configuration tool from the official website and copy it to the folder where the exclusion item was added in the previous step.
  • Step 4: reopen the real-time protection and cloud provided protection in step 1.
